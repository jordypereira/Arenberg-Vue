# Arenberg-Vue

Local install:
- git clone 
- cd Arenberg-Vue
- yarn install

Development:
- All pages are in /src/views
- All components in /src/components
- All Scss in /src/assets/scss
- All images in /public/images
- Global components (Navbar and footer) Are in the App.vue file

- yarn serve 

Production:
- yarn build
